# ✈️ Trip Expense

A full-stack trip expense tracker with Python/Flask backend and SQLite database.

## Project Structure

```
tripexpense/
├── app.py          ← Flask backend + SQLite database
├── index.html      ← Frontend (served by Flask)
├── start.sh        ← Quick start script
└── README.md
```

## Requirements

- Python 3.8+
- Flask  →  pip install flask

## How to Run

```bash
python3 app.py
```

Then open: http://localhost:5000
